## Ship the agent your product deserves.

Vendo is an agent that lives inside your product. It calls your API as the
signed-in user, builds views your users own, and runs pre-approved
automations, all rendered inside your brand and your guardrails.

## The path to production

<Steps>
  <Step title="Connect your app">
    Run `npx vendo init` to extract your host API into guarded tools. See
    [Connect your app](/connect/vendo-init).
  </Step>
  <Step title="Make it yours">
    Theme generated UI to match your brand. See
    [Make it yours](/connect/theming).
  </Step>
  <Step title="Give it reach">
    Let users connect their own tools and accounts. See
    [Give it reach](/capabilities/connected-accounts).
  </Step>
  <Step title="Ship it">
    Deploy with the same Postgres schema locally and in production. See
    [Ship it](/deploy/deploying).
  </Step>
</Steps>
