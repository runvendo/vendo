Vendo puts an agent inside your product. It can converse, call your APIs as the
signed-in user, build user-owned apps, and run pre-approved automations.

```bash
npm install @vendoai/vendo
npx vendo init
```

`@vendoai/vendo` supplies the default composition. The `vendoai` package is a
thin alias. Every block is also available à la carte.

## Default behavior

- PGlite persists at `.vendo/data`.
- The host supplies one AI SDK `LanguageModel`.
- Every request resolves a host principal; `null` becomes an ephemeral session.
- Every tool path passes through guard policy and audit.
- Tree apps work without a sandbox. A sandbox adapter unlocks server-backed apps.
- Chrome shows a loud notice when no policy or judge is configured.

Cloud-gated sharing, publishing, org overlays, and pinning activate with
`VENDO_API_KEY`. The rest of the stack is complete in the open-source packages.
