# Automation Created Preview

Real Vendo shell preview for the automation approval and created-to-running notification.

Run it from the repo root:

```sh
pnpm --filter demo-accounting exec vite ../../previews/automation-card-real --host 127.0.0.1 --port 5179
```

Open `http://127.0.0.1:5179/`, then click `Play animation` for a deterministic filming take.
